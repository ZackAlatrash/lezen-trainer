# Lezen Trainer

Installable, offline practice app for the NT2 Staatsexamen **Lezen I** reading exam.

Open the live app: **https://zackalatrash.github.io/lezen-trainer/**

On Android (Chrome): open the link → menu (⋮) → **Add to Home screen / Install app**.
Works fully offline after the first load. Built from past public exams (CvTE), for personal study.
